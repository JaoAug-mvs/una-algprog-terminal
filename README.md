⚡ Comandos mais ultiloizados:
Se encontram códigos mais ultilizados:

- `cd`: Para navegar entre pastas.
- `dir`: Para listar arquivos.
- `mkdir`: Para a criação de pastas
- `type nul > textão.txt`: Para a criação de arquivos de texto
- `Z:`: Para mudar o diretório de armazenamento, pois eu uso o meu 2° diretório para projetos
